# TO_ARM
### : 이 빌드 결과물은 실행을 위해 만들어졌습니다
  - Dialogue system 구현
    : 다이얼로그 시작 버튼,
      한 글자씩 써지는 텍스트 효과,
      유니티 종료 버튼
<설명>
  1. 다이얼로그 시작 버튼 누르면 시작
  2. 한 글자씩 써지는 텍스트 효과
  3. 화면 클릭 시 모든 텍스트 보여주기 or 다음 텍스트 보여주기
  4. 종료 버튼 누르면 프로그램 종료
